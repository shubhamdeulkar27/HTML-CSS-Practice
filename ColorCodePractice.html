RGB

This is Blue Color.

RGBA

This is and RGBA Color Which has A as Alpha to specify Opacity.

HEX

This is HEX Coolr Code That is Heaxdecimal Code in which color code is like #rrggbb, where rr,gg,bb ranges from 00 to ff.

HSL

This is HSL Color code i.e. Hue Saturation AND Lightness. Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue. Saturation is a percentage value, 0% means a shade of gray, and 100% is the full color. Lightness is also a percentage value, 0% is black, and 100% is white.
